The Ultimate JavaScript Framework, Toolkit, and Library

Toolkit

JSTotal provides a toolkit that can carry out any Mathematical, currency formatting, time formatting, form validating, or animation function that you could ever want with a single function call.

Library

JSTotal is a huge library of cross-platform and cross-device JavaScript functionality. However, it is compressed to a remarkably small 895kb.

Framework

JSTotal provides a modern standards-compliant framework for developing Single-Page Applications as well as responsive websites.
